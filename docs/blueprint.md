# **App Name**: ScholAI

## Core Features:

- Intelligent Study Planner: An AI tool that generates personalized learning paths and schedules based on individual student performance and learning gaps.
- Automated Grading Assistant: An AI tool used by teachers to perform semantic logic analysis and provide initial feedback on open-ended student assignments.
- Role-Based Authentication: Secure sign-in for Administrators, Teachers, and Students using Firebase Authentication for tailored access to the portal.
- Student Data Registry: A centralized Firestore database used to store student transcripts, contact information, and academic histories securely.
- Real-time Course Management: Dashboard for teachers to create classrooms, assign curriculum materials, and update student records in real-time.
- Predictive Performance Dashboard: A visualization tool showing student progress trends with AI-calculated predictions of upcoming grade outcomes.
- Instant Resource Generator: An AI tool that converts uploaded syllabus documents into bite-sized summaries and practice quizzes for students.

## Style Guidelines:

- Primary Color: Refined Academy Indigo (#3152A5) conveying trust and scholarly intelligence.
- Background Color: Crisp Arctic Blue (#F2F5FB), a clean, modern canvas that provides a focused learning environment.
- Accent Color: Knowledge Teal (#146D86), providing sharp contrast for high-priority calls to action.
- Headline font: 'Literata' (Serif) for an academic and formal feel. Body font: 'Inter' (Sans-serif) for high readability in functional data-rich areas.
- Minimalist, mono-line icons using a medium stroke weight to maintain a professional yet approachable aesthetic.
- A sidebar-focused navigation structure to maximize screen space for complex data tables and lesson views.
- Progressive, ease-out transitions for dashboard loading and subtle micro-interactions on task completion icons.
