# Preview Project Workflow

Follow this guide to start the local Next.js development server and preview the Axora application.

## Steps

### 1. Start Local Server
Run the Next.js development server locally using Turbopack on port 9002:
```bash
npm run dev
```

### 2. Verify URL
Open the browser at [http://localhost:9002](http://localhost:9002) and verify that the page renders correctly without console errors.
